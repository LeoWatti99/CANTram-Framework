◆ addChannel()

virtual bool PWMCore::addChannel ( OutputDefinition * def)
pure virtual

Add/allocate a PWM channel for the given output definition.

Reserve any internal timer/channel resources required to drive PWM on the provided OutputDefinition. The implementation should validate that the OutputDefinition supports PWM and return true on success.

Parameters
defPointer to the OutputDefinition to register for PWM.
Returns
true if the channel was added/allocated successfully, false otherwise.

Implemented in ESP32PWMCore.

Set PWM duty cycle for a registered output.

Drive the output described by def with the specified duty value. Duty units are implementation-specific but are expected to be in the range 0..4095 for 12-bit resolution.

Parameters
defPointer to the OutputDefinition to set PWM on.
dutyDuty value to apply (e.g., 0..4095 for 12-bit resolution).
