\doxysubsection{Detailed Description}
Abstract base for UART hardware cores and provider interface. 

Define the abstract \doxylink{class_u_a_r_t_core}{UARTCore} class which concrete platform implementations must inherit. Provides methods for installation, configuration, pin setup and data IO. Also defines the \doxylink{class_u_a_r_t_provider}{UARTProvider} interface used to obtain \doxylink{class_u_a_r_t_core}{UARTCore} instances.
