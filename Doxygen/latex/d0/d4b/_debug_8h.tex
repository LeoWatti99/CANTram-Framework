\doxysubsection{Detailed Description}
Logging and debug macro definitions for \doxylink{class_c_a_n_tram_core}{CANTram\+Core}. 

Defines macros for debug, info, error, warning, developer error and chip-\/level logging. Macros prepend a level prefix and optionally include file/line information when DEBUG\+\_\+\+SHOW\+\_\+\+LOCATION is defined. Also provides a NULL pointer check macro that prints a critical error and halts execution when a null pointer is detected.
