\doxysubsection{Detailed Description}
Abstract interface for providers that control outputs. 

Implementors provide mechanisms to configure outputs (mode), set output state and read state. Concrete providers adapt MCU GPIO, expanders, or shift registers to the \doxylink{class_output_definition}{Output\+Definition} API.
