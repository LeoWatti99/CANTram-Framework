\doxysubsection{Detailed Description}
Base class for hardware resources. 

Defines the abstract base class for hardware resources (UART, SPI, I2C, PWM) used by the \doxylink{class_c_a_n_tram_core}{CANTram\+Core}. Provides usage counting and basic lifecycle methods that concrete hardware resource implementations should override.
