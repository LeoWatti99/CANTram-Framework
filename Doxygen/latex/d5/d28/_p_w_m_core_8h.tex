\doxysubsection{Detailed Description}
Abstract PWM core interface for CANTram. 

Declares the \doxylink{class_p_w_m_core}{PWMCore} base class used to provide PWM functionality to modules. Platform-\/specific implementations should derive from this class and implement channel allocation and duty control. \doxylink{class_p_w_m_core}{PWMCore} is also a \doxylink{class_hardware_resource}{Hardware\+Resource} and participates in the \doxylink{class_c_a_n_tram_core}{CANTram\+Core} hardware resource lifecycle.
