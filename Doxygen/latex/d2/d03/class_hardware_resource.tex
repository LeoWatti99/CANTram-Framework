\doxysubsection{Detailed Description}
Base class for hardware resources like UART, SPI, I2C, PWM, etc. 

\doxylink{class_hardware_resource}{Hardware\+Resource} provides a common interface for resources managed by \doxylink{class_c_a_n_tram_core}{CANTram\+Core}. It implements usage counting and default reset/usage behaviour. Derived classes must implement \doxylink{class_hardware_resource_ad1058e4c12406f756755fad5db55dc77}{get\+Type()} and may override \doxylink{class_hardware_resource_a6b7a709561faf5d344b7c3e14bf37558}{get\+Max\+Usages()} and \doxylink{class_hardware_resource_a7372684f2afb5139ef8464194a087649}{reset()} as required.
