It resets the internal state of the core, clears the module and hardware resource lists, and resets all attached modules and hardware resources. It should be called to reinitialize the core to a clean state. Normally this function should only be used during testing, but not in production code.

\begin{DoxyReturn}{Returns}
true if the reset was successful, false otherwise
