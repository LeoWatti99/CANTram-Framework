Detailed Description

Describes an available output (GPIO or shift register bit).

Holds metadata about an output such as whether it is on a shift register, the pin/bit identifier, and capabilities (ADC, PWM, input-only). Also stores a pointer to the OutputProvider responsible for configuring/controlling the physical output.
