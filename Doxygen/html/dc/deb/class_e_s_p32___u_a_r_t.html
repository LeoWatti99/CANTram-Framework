bool ESP32_UART::applyConfig ( )

Configure UART communication parameters.

Applies baudrate, data bits, parity and stop bits to the UART peripheral. Requires the instance to be in INSTALLED state prior to calling.

Parameters
baudrateDesired baud rate (e.g., 115200).
dataBitsData bits selection from UARTCore::DataBits.
parityParity selection from UARTCore::Parity.
stopBitsStop bits selection from UARTCore::StopBits.
Returns
true if configuration succeeded, false otherwise.
