Detailed Description

Abstract PWM core interface for CANTram.

Declares the PWMCore base class used to provide PWM functionality to modules. Platform-specific implementations should derive from this class and implement channel allocation and duty control. PWMCore is also a HardwareResource and participates in the CANTramCore hardware resource lifecycle.
