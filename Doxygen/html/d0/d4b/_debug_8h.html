Detailed Description

Logging and debug macro definitions for CANTramCore.

Defines macros for debug, info, error, warning, developer error and chip-level logging. Macros prepend a level prefix and optionally include file/line information when DEBUG_SHOW_LOCATION is defined. Also provides a NULL pointer check macro that prints a critical error and halts execution when a null pointer is detected.

#define NULL_PTR_CHECK ( ptr)

}

Check a pointer for null and report a critical error if null.

Evaluates the provided pointer expression and if it is nullptr, prints a critical error with the pointer name and halts execution.

Parameters
ptrPointer expression to check.
Returns
This macro does not return on failure; execution halts in an infinite loop after reporting the error.
