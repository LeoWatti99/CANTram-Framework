bool BusModuleV1_0::requestGPIOs ( )
overridevirtual

Request GPIO definitions from CANTramCore and validate pins for module interfaces.

Retrieves UART RX/TX and SPI CS output definitions from the core, validates they are not on shift registers and logs the assigned pins. Invalid pins will cause related interfaces to be invalidated.

Returns
true if all required GPIO definitions were found and valid, false otherwise.
