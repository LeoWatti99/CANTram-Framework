Detailed Description

Abstract base for UART hardware cores and provider interface.

Define the abstract UARTCore class which concrete platform implementations must inherit. Provides methods for installation, configuration, pin setup and data IO. Also defines the UARTProvider interface used to obtain UARTCore instances.
