# CAN-TRAM-Framework

Das CAN-TRAM-Framework ist ein mikrocontrollerbasiertes, modulares Steuerungssystem, dass sich aus einzelnen Modulen zusammensetzen lässt. Die Struktur des Systems orientiert sich an üblichen SPS-Systemen, sodass eine CPU bzw. ein Hauptmodul um beliebig viele Module erweitert werden kann, um Funktionen hinzuzufügen.

## Inventor CAD Dateien öffnen
Die CAD Dateien des Projekts sind im Ordner [Inventor CAD](https://github.com/LeoWatti99/CANTram-Framework/tree/main/Inventor%20CAD)) enthalten. Um das Projekt wie vorgesehen zu öffnen muss in Inventor 2025 oder höher die Datei [Controller.ipj](https://github.com/LeoWatti99/CANTram-Framework/blob/main/Inventor%20CAD/Controller.ipj) geöffnet werden.

## KiCAD Schaltungen öffnen
Im Ordner [KiCad Schaltungen](https://github.com/LeoWatti99/CANTram-Framework/tree/main/KiCad%20Schaltungen) sind alle Projekte der einzelnen Platinen enthalten. Darin befinden sich sowhol Prototypen Schaltungen als auch aktuelle Modulplatinen. Um die Projekt zu öffnen wird KiCad 9.0.6 benötigt. über die .kicad_pro-Datein in den Unterordnern können die Projekte vollständig mit der Verlinkung zwischen Schaltplan und Layout geöffnet werden.

## C++ Framework
Die Firmware des Systems ist als C++ Bibliothek im Ordner [Framework](https://github.com/LeoWatti99/CANTram-Framework/tree/main/Framework) enthalten.
