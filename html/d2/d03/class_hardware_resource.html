Detailed Description

Base class for hardware resources like UART, SPI, I2C, PWM, etc.

HardwareResource provides a common interface for resources managed by CANTramCore. It implements usage counting and default reset/usage behaviour. Derived classes must implement getType() and may override getMaxUsages() and reset() as required.

Request a usage slot for this resource.

Increments the internal usage counter if the resource has remaining capacity. If the resource is already at its maximum usages, the request is rejected and an error is logged.

Returns
true if the usage request was granted, false if the resource is already fully used.
