bool ESP32_CANCore::begin ( )
overridevirtual

Initialize the CAN interface.

This function first checks if the interface is already initialized. If not, it prepares the general, timing, and filter configurations based on the stored settings. It then installs the TWAI driver, configures alerts, and starts the driver.
