bool CANTramModule::attachModule ( uint8_t slot,
uint8_t gpioStart )
virtual

Attach this module to a slot and assign GPIO range.

Attach this module to a slot and assign its GPIO range.

Initializes the module's internal SLOT and GPIO_START and triggers GPIO/resource provisioning and interface registration sequences.

Parameters
slotSlot index to attach the module to.
gpioStartStarting GPIO index assigned to this module.
Returns
true if attachment and provisioning succeeded, false otherwise.

Sets the module's SLOT and GPIO_START values, logs the attachment and runs the standard initialization sequence: provideGPIOs, provideHardwareResources, requestGPIOs, requestHardwareResources and addInterfaces. Logs success or warnings based on the combined result.

Parameters
slotSlot index to attach the module to.
gpioStartStarting GPIO index assigned to this module.
Returns
true if all initialization steps succeeded, false if any step failed.

Reimplemented in AnalogModuleV1_0, BusModuleV1_0, DigitalModuleV1_0, MainModuleV1_0, and RelaisModuleV1_0.
